# SE4352-Assignment2-Refactoring

In this homework, you are given the program to support the management of a movie rental place as in Homework 1. 

You are required to perform refactoring on that program to improve its quality. You are encouraged to use refactoring services in IDEs. Then, you are required to 

1) add a main() method to test the program; and  

2) add a new method to print the statement for a customer in XML format, e.g., <name> John Smith </name>, <movie> Independent Day </movie>, etc. 

Please submit your resulting code and a short Word document explaining the rationales behind your refactoring changes. Your solution must at least contain: 

1. At least 3 method extraction operations 
2. At least 3 new classes 
3. At least 3 moving method operations 
4. At least 3 renaming operations 
5. 1-2 replacements of data types 

You are free to add more operations as you like
